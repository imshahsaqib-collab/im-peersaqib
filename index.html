Getting Started with the Internet of Things

This book covers the fundamentals of IoT: sensors, connectivity (Wi-Fi, Bluetooth, LoRa), microcontrollers (ESP32/Arduino), and building simple cloud dashboards. Great for beginners who want a practical approach to connecting devices safely.

Why read it?

Hands-on projects, clear diagrams, and real-world examples for deploying small-scale IoT systems.

Practical Electronics for Inventors

An in-depth reference for hobbyists and students. It explains real components, circuit design practices, and practical methods used in hardware prototyping and debugging.

Useful for bridging theory and hands-on circuit-building skills.

Computer Hardware: A Systems Approach

Explores how modern computer systems are built. From the CPU internals to memory hierarchies and I/O ‚Äî an excellent resource for understanding the hardware side of computing.

Great for learners who want to understand how software maps to hardware.

TinyML: Machine Learning with TensorFlow Lite on Microcontrollers

TinyML focuses on running ML models on microcontrollers. Learn model quantization, inference on-device, and power-efficient deployment strategies for embedded ML tasks.

Recommended for applying ML on resource-constrained hardware.

The Hardware Hacker's Handbook

A practical guide for reverse-engineering, interfacing hardware, and designing custom PCBs for maker projects.

Perfect for hands-on enthusiasts and small project designers.

IoT Security ‚Äî Building Secure Connected Devices

Security best practices for IoT devices: secure boot, encryption, credential management, and safe update mechanisms.

Important for anyone deploying devices beyond the lab environment.
